# 🍽️ Zomato Restaurant Data Analysis & Business Insights

## 📌 Project Overview

As a Data Analyst, the objective of this project was to analyze the Zomato Restaurant Dataset, uncover meaningful patterns, visualize trends, and provide actionable business recommendations based on real-world restaurant data.

The project follows a complete Data Analytics workflow including data loading, cleaning, exploratory data analysis, visualization, and business insight generation.

## 🎯 Problem Statement

Analyze a real-world restaurant dataset to:

- Understand the data structure and quality
- Clean and preprocess the dataset
- Identify trends and patterns
- Create meaningful visualizations
- Generate business recommendations based on findings

## 🛠️ Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📂 Dataset Information

Dataset: Zomato Restaurant Dataset

The dataset contains information about:

- Restaurant Names
- Countries
- Cities
- Cuisines
- Ratings
- Price Range
- Votes
- Online Delivery Availability
- Table Booking Availability
- Restaurant Types

## 📊 Data Analysis Performed

### Step 1: Data Inspection
- Loaded dataset using Pandas
- Checked data types
- Identified missing values
- Checked duplicate records

### Step 2: Data Cleaning
- Handled missing values
- Merged country information using Country Code
- Removed irrelevant columns
- Corrected data types where necessary

### Step 3: Exploratory Data Analysis
Answered key business questions including:

- Which countries have the highest number of restaurants?
- Which cuisines are most popular?
- How does online delivery impact ratings?
- Which cities have the most restaurants?
- How does price range affect customer ratings?

## 📈 Visualizations Created

- Bar Chart
- Line Chart
- Histogram
- Scatter Plot
- Pie Chart
- Correlation Heatmap

Each visualization was created with proper titles, labels, and business interpretation.

## 🔍 Key Insights

1. Restaurant distribution is highly concentrated in specific countries.
2. Restaurants offering online delivery generally receive better customer ratings.
3. Mid-priced restaurants tend to achieve higher customer satisfaction.
4. Popular restaurants with more votes often maintain better ratings.
5. A few cuisines dominate customer preferences across locations.

## 💡 Business Recommendations

- Expand operations in underrepresented regions.
- Encourage restaurants to provide online delivery services.
- Focus on high-demand cuisines when planning expansion.
- Improve customer engagement to increase ratings and visibility.
- Analyze customer preferences to optimize pricing strategies.
